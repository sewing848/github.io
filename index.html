2. Message Preparation

4. Block Formation

4.1. Group encoded words into proto-blocks

↓

5. Block Ordering

Add order information to blocks

↓

6. Encryption

6.1. Perform Fisher-Yates shuffle on dictionary codes

6.2. Substitute shuffled codes

6.3. Apply modular arithmetic cipher

↓

7. Final Output

Return array of encrypted message blocks as strings
